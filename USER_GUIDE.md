## Student Guide

### Getting Started

1. **Login**
   - Go to the login page
   - Enter your email and password provided by admin
   - Click "Sign In"

2. **Dashboard**
   - View your attendance percentage
   - See upcoming tests
   - Check recent notes and homework
   - View important announcements

### Viewing Content

1. Navigate to **Notes** from the sidebar
2. Select your batch and subject
3. Click on any content to view/download
4. PDFs can be downloaded for offline access

### Taking Tests

1. Go to **Tests** from the sidebar
2. Click on an available test
3. Read instructions carefully
4. Answer all questions
5. Submit before time expires
6. View results after submission

### Submitting Homework

1. Go to **Homework** from the sidebar
2. Click on an assignment
3. Download the homework file
4. Complete the assignment
5. Upload your submission
6. Check status (Pending/Checked)

### Viewing Attendance

1. Go to **Attendance** from the sidebar
2. View your attendance percentage
3. See attendance history by date
4. Check monthly summary

### Profile

1. Click on your name in the navbar
2. View your profile information
3. See your batch and academic year

## Teacher Guide

### Getting Started

1. **Login**
   - Use credentials provided by admin
   - Access teacher dashboard

2. **Dashboard**
   - View assigned batches
   - See upcoming tests
   - Check pending homework reviews
   - View statistics

### Uploading Content

1. Go to **Content** → **Upload**
2. Select batch and subject
3. Enter chapter name
4. Choose content type (PDF/Image/Video)
5. Upload file or paste YouTube URL
6. Add description (optional)
7. Click "Upload"

### Creating Tests

1. Go to **Tests** → **Create Test**
2. Enter test details:
   - Title
   - Subject
   - Duration
   - Start/End dates
3. Add questions:
   - Select question type (MCQ/Short Answer)
   - Enter question text
   - Add options (for MCQ)
   - Set correct answer
   - Assign marks
4. Save test

### Marking Attendance

1. Go to **Attendance**
2. Select batch and date
3. Mark students as Present/Absent
4. Save attendance

### Reviewing Homework

1. Go to **Homework**
2. Click on an assignment
3. View student submissions
4. Download submissions
5. Mark as "Checked" when reviewed

### Viewing Reports

1. Go to **Attendance Reports**
2. Select batch and date range
3. View attendance statistics
4. Export reports if needed

## Admin Guide

### Getting Started

1. **Login**
   - Use admin credentials
   - Access admin dashboard

2. **Dashboard**
   - View system statistics
   - See active students/teachers
   - Monitor recent activities

### Managing Students

1. Go to **Students**
2. **Add Student:**
   - Click "Add Student"
   - Enter student details
   - Assign to batch
   - Set password
   - Save
3. **Edit Student:**
   - Click on student
   - Update information
   - Change batch if needed
4. **View Student:**
   - Click on student name
   - View profile and statistics

### Managing Batches

1. Go to **Batches**
2. **Create Batch:**
   - Click "Create Batch"
   - Enter batch name
   - Select academic year
   - Assign subjects
   - Assign teachers
   - Save
3. **Edit Batch:**
   - Click on batch
   - Update information
   - Manage students/teachers

### Managing Teachers

1. Go to **Teachers**
2. **Add Teacher:**
   - Click "Add Teacher"
   - Enter teacher details
   - Assign to batches
   - Set password
   - Save
3. **Edit Teacher:**
   - Click on teacher
   - Update information
   - Manage batch assignments

### Managing Subjects

1. Go to **Subjects**
2. **Create Subject:**
   - Click "Create Subject"
   - Enter subject name
   - Add description
   - Save
3. Assign subjects to batches

### Managing Academic Years

1. Go to **Academic Years**
2. **Create Academic Year:**
   - Enter year (e.g., 2024-25)
   - Set start/end dates
   - Set as active if needed
3. **Set Active Year:**
   - Only one year can be active
   - Active year is used for new batches

### Creating Notices

1. Go to **Notices**
2. Click "Create Notice"
3. Enter:
   - Title
   - Content (English and Assamese)
   - Type (General/Exam Date/Holiday/Important)
   - Priority
   - Expiry date
   - Target batch (optional)
4. Save notice

### Viewing Reports

1. Go to **Reports**
2. **Attendance Reports:**
   - Select batch/student
   - Choose date range
   - View statistics
   - Export if needed
3. **Performance Reports:**
   - View test results
   - See student rankings
   - Analyze performance

### System Settings

1. Manage academic years
2. View system statistics
3. Monitor user activity
4. Configure system settings

   - Post important notices

## Troubleshooting

### Login Issues
- Check email and password
- Contact admin if locked out
- Clear browser cache

### Upload Issues
- Check file size (max 10MB)
- Verify file format
- Try again after some time

### Display Issues
- Refresh page
- Clear browser cache
- Try different browser
